# Monster-Identifier-Project
A simple CNN model used to identify different monsters/creatures from various sources such as Pokémon and Digimon. This was my final project for a Deep Learning course.

There are three scripts here:
MonsterIdentifier: The primary notebook that outlines the process for creating a CNN model to identify the various creatures.
WebScraper: Code used for web scraping images from various Wikipedia pages.
Gen: A bonus notebook used to create a simple GAN model to transform Digimon creatures into a more Pokémon-style appearance.

The images are not included here due to their large file size. However, the images can be obtained from the Kaggle dataset sources listed below or by using the web scraper script.
